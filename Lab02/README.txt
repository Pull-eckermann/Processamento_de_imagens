Segundo trabalho de laboratório da disciplina de Processamento de imagens

Código escrito em Python3, utilizando as seguintes bibliotecas:
- Open-cv (cv2)
- sys
- numpy

O código está contido inteiramente no arquivo floresta.py

Antes de executar, certifique que o arquivo das imagens estejam no mesmo diretório que floresta.py, pois o código lê as imagens à partir do diretório corrente.

Para compilar e executar o programa, basta utilizar o comando python3 com o nome do arquivo, seguido pela imagem a ser segmentada e o nome da imagem de saída, da seguinte maneira:

$ python3 floresta.py <IMG ENTRADA> <IMG SAIDA>

ex: python3 floresta.py f1.png out.png

O programa criará uma imagem com o nome expecificado, sendo essa a imagem segmentada com as áreas de floresta.
